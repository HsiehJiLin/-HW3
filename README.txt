
1. Your game client uses pygame, so install it before running:

	### Windows / macOS / Linux:
    		pip install pygame


2. start lobby client
	### Windows (PowerShell)
		py client.py

	### macOS/Linux
		python3 client.py

3. start Developer Client
	### Windows (PowerShell)
		py Developer_client.py

	### macOS/Linux
		python3 Developer_client.py
